# Developing a RESTful API With Node and TypeScript

## Dependency
- `npm install -g gulp`

## Command
1. Install dependencies - `npm install`
1. Compile - `npm run serve`
1. Test - `npm test`

## Api endpoint
- localhost:3000/api/dbs/

- [Simple whois REST API](https://github.com/wingsuitist/whoissv)
